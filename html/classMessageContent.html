Detailed Description

This type of code to parse a Whiteboard message in a call-back is everywhere in the MiPal code. This is an attempt to put it in one single place. The interface to the Whietbaord needs re-factoring and built into a framework where functionality like this to parse the type and content of a message in a call-backs is simplified.
